- abstract
가상화폐가 등장하면서, 많은 사람들은 자신의 컴퓨터에서 채굴 프로그램을 사용하여 가상화폐를 채굴하기 시작했다.
하지만 사람들은 자신의 컴퓨터 자원을 이용한 채굴 속도에 만족을 못하고 다른 방법을 생각하였는데, 웹에서 다른 사람들의 컴퓨터 자원을 이용하여 가상화폐를 채굴하는 coinhive 같은 서비스 사이트들이
생기기 시작하였다.
이러한 방법을 이용하여, 웹 관리자들은 광고를 대신하여 웹 마이닝 코드를 넣거나, 웹의 빽그라운드에서 웹 마이닝 코드가 동작하게 끔 웹 페이지를 구성한다.
이러한 사이트의 방문자들은 자신의 컴퓨터 자원을 웹 관리자가 가상화폐 채굴을 하는데 사용한다는 것을 쉽게 알지 못한다.
이에, 다른 사용자의 컴퓨터 자원을 사용하여 가상화폐를 채굴하는 것이 도덕적으로 문제가 있고, CPU, GPU 등이 과하게 사용되어 사용자의 컴퓨터에 안좋은 영향을 준다고 생각하였다.
그러면서 이를 막아주는 여러 안티 마이닝 프로그램이 등장하였다.
하지만 대다수의 마이닝 코드 디텍트 프로그램은 블랙리스트에 존재하는 특정 도메인에서만 자바스크립트를 실행 방지하는 방법으로만 마이닝 코드로부터 컴퓨터를 보호한다.
이런 방법은 대부분이 우회가 가능하기 때문에 실질적으로 마이닝 코드를 막기 힘들다.
따라서 우리는 기존에 존재하는 자바스크립트 웹 기반 마이닝 코드들에 대해 분석을 수행하고, 특정 엘리먼트들을 분류할 것이다.
또한 기존에 존재하는 안티 마이닝 프로그램들을 분석하고, 블랙리스트와 CPU 사용량 체크 모듈을 발전시킬 것이다.
따라서 우리는 동적 분석 및 여러 기술을 이용하여 최대한 많은 마이닝 코드를 막을 수 있게 하는 프로그램을 개발할 것이다.

As a cryptocurrency comes up, a lot of people started cryptocurrency mining by using mining programs in their computers.
More and more, the people thought a way how to mine with faster speed than the local computer speed.
So, services like CoinHive are made, many sites have consisted of web-based mining codes to use a visitor's computer resources.
Using these services, web administrators involved a javascript mining code instead of advertisement or hide a javascript mining code to run in background.
Visitors can't know that web administrators use their computer resources to mine cryptocurrencies in visitors' computer.
Because some people thought that the web mining services are morally problematic, and has a bad effect on visitors' computers by using CPU, GPU and so on.
For that reason, several programs were made to detect and prevent these services from mining.
However, most of the programs detect javascript mining code based on blacklist which have some specific domain.
This way is practically impossible to detect mining codes, because most of web administrators can bypass a anti mining programs.
So, we will analyse web-based javascript mining codes and classification the result of a analysis.
We will also analyse anti mining programs and enhance a blacklist and a CPU checking module.
Finally, we will use a javascript event logger to detect mining code.
So, we suggest a practical program to prevent mining code from working on web browsers and a way to detect malicious codes.
